# Overview  
### Smart Embedded Control & Sensing Platform

The **Smart Embedded Control & Sensing Platform** is a compact, feature-rich controller board designed to demonstrate professional electronic hardware design practices. The board integrates sensing, motor control, audio functionality, and multi-protocol communications into a single embedded platform.

This design was completed during my training in the **Complete Electronics Hardware Design Course 2025 ‚Äì EsteemPCB**, where I practiced requirements analysis, schematic design, PCB layout, component selection, and system documentation using industry-standard workflows.

---

## ‚ú® Key Capabilities

- **High-performance ARM Cortex-M4 Processor** (STM32F407 @ 168 MHz)  
- **On-board ST-Link Debugger** (STM32F103-based)  
- **Ethernet, USB 2.0, CAN, UARTs** for hardware connectivity  
- **Precision 24-bit ADC** for microvolt-level measurements  
- **Dual Motor Control** (12V / 3A / 36W per motor)  
- **Audio Subsystem** with DAC, speaker amps, and microphone input  
- **4-layer PCB** with controlled impedance routing  
- **Robust power system** with reverse polarity and ESD protection
